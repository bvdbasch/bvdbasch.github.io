Strifecake: a bloggy Anchor CMS theme

December 18, 2021 by Bonno van der Basch

WARNING: some skills may be required to run this theme to full satisfaction‚Ä¶ This is NOT a download and run type deal‚Ä¶

This is the result of a week of development and was incredibly fun to do! The work is open source and available in this github repo

A couple of custom php functions had to be created in order to make this all work. This is due to the limitations the current stable version of Anchor CMS has. The functions mainly deal with parsing article details from specific categories.

The theme works based on php include blocks to allow for easy customization.
